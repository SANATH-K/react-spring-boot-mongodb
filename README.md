# react-spring-boot-mongodb
# Requirements:
  ## 1.Java -1.8.x
  ## 2.Maven -3.8.x
  ## 3.MongoDB -4.4.x
# Steps:
 ## 1.Download the react-spring-boot-mongodb repository
 ## 2.Open terminal
      cd backend
      mvn spring-boot:run
 ## 3.open another terminal
      cd frontend
      npm install 
      npm start
